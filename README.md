🚀 AI for Bharat - Code Guru
Code Guru is an AI-powered code review and optimization platform designed specifically for the Indian developer ecosystem. It leverages Large Language Models (LLMs) to provide intelligent feedback, and performance optimization suggestions for various programming languages.

🛠 Features
Intelligent Code Review: Automatically scans code snippets for logic errors and anti-patterns.

Language Localization: Designed to support documentation and explanations in multiple Indian languages (powered by AI4Bharat models).

One-Click Refactoring: Provides "Guru Suggestions" to rewrite messy code into clean, maintainable blocks.

🏗 Tech Stack
Frontend: React.js
Styling: Tailwind CSS
Authentication: NextAuth.js
AI Integration: AWS Bedrock and OpenAI API
Deployment: Vercel

🚦 Getting Started
Prerequisites
1. Node.js (v18 or higher)
2. npm / yarn / pnpm

Installation
 1. Clone the repository:
git clone https://github.com/username/aiforbharatcodeguru.git
cd aiforbharatcodeguru

2. Install dependencies:
npm install
Set up Environment Variables:
Create a .env.local file in the root directory and add your keys:

Bash
npm run dev
